# Необходимо изменить код лабораторной работы
(Реализовать три класса, представляющих сведения об объекте. Один класс – базовый, представляющие сведения о группе устройств. Еще два класса – производные, представляющие более конкретные типы устройств.
В программе формируется список, в который можно включать сведения об устройствах разных типов. Пользователь может вывести сведения обо всех устройствах, добавить или удалить сведения об устройстве.) по принципам SOLID
